# Git Temporal

## Run the alpha POC

### Clone the repo and install dependencies

```bash
git clone https://github.com/git-temporal/git-temporal.git
cd git-temporal
npm install
```

### Start the log scraping service

```bash
git clone https://github.com/git-temporal/git-temporal.git
cd git-temporal/packages/api-node-express
```
